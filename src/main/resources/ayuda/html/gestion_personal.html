Gestión de Trabajadores

Esta ventana permite realizar el mantenimiento de los trabajadores del sistema. Desde aquí podrás insertar, modificar y eliminar trabajadores, así como consultar los datos existentes mediante la tabla inferior.

Campos del formulario

Campo Descripción
Nombre del trabajador Nombre completo del trabajador.
Campo obligatorio.
Documento Documento identificativo del trabajador.
Puede ser NIF, NIE, CIF o Libre.
El sistema aplica automáticamente el formato según el tipo seleccionado.
Tipo Indica el tipo de trabajador:
  • Administración
  • Servicios
Es obligatorio seleccionar uno de los dos.
PVP Hora Precio por hora del trabajador.
Se introduce mediante un control numérico.
Población Localidad de residencia del trabajador.

Botones disponibles

  • Insertar: Añade un nuevo trabajador con los datos introducidos.
  • Modificar: Actualiza los datos del trabajador seleccionado.
    Para modificar, realiza doble clic sobre un trabajador en la tabla.
  • Eliminar: Elimina el trabajador seleccionado del sistema.
    No se permitirá la eliminación si el trabajador tiene servicios asociados.
  • Cancelar: Limpia los campos y restaura el formulario a su estado inicial.

Tabla de Trabajadores

En la parte inferior de la ventana se muestra una tabla con todos los trabajadores registrados.
Para editar o eliminar un trabajador, haz doble clic sobre la fila correspondiente.

Notas importantes

  • Todos los campos obligatorios deben estar correctamente rellenados.
  • El documento debe estar completo y respetar el formato indicado.
  • Las operaciones de eliminación requieren confirmación previa.
