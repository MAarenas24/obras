Gestión de Obras

Esta ventana permite realizar el mantenimiento de las obras registradas en el sistema. Desde aquí podrás insertar, modificar y eliminar obras, así como consultar las existentes mediante la tabla inferior.

Campos del formulario

Campo Descripción
Código de Obra Identificador único de la obra.
No puede modificarse una vez creada la obra.
Descripción Breve descripción o localización de la obra.
Fecha de Finalización Fecha prevista de finalización de la obra.
El formato debe ser DD/MM/YYYY.
Este campo puede dejarse vacío si no se conoce la fecha.

Botones disponibles

  • Insertar: Registra una nueva obra con los datos introducidos en el formulario.
  • Modificar: Actualiza los datos de una obra existente.
    Para modificar una obra, haz doble clic sobre ella en la tabla.
  • Eliminar: Borra la obra seleccionada.
    No se permitirá la eliminación si la obra tiene servicios asociados.
  • Cancelar: Limpia los campos del formulario y restablece el estado inicial.

Tabla de Obras

En la parte inferior de la ventana se muestra una tabla con todas las obras registradas.
Para editar o eliminar una obra, realiza doble clic sobre la fila correspondiente.

Notas importantes

  • Todos los campos obligatorios deben rellenarse antes de insertar o modificar.
  • Las fechas deben respetar el formato indicado.
  • Las operaciones de eliminación requieren confirmación.
