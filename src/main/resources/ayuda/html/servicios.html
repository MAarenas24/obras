Registro de Servicios

Esta ventana permite registrar y gestionar los servicios realizados por un trabajador en una obra concreta, indicando la fecha y el horario trabajado.

Cada servicio queda asociado al trabajador actualmente identificado en el sistema.

Campos del formulario

Campo Descripción
Trabajador Muestra el nombre del trabajador autenticado.
Este campo es informativo y no puede modificarse.
Obra Obra en la que se ha realizado el servicio.
Se selecciona desde una lista desplegable.
Fecha Fecha en la que se realizó el servicio.
El formato es YYYY/MM/DD.
El sistema aplica una máscara de entrada automática.
Hora de inicio Hora y minuto de inicio del servicio.
Se introduce mediante controles numéricos.
Hora de fin Hora y minuto de finalización del servicio.
Debe ser posterior a la hora de inicio.

Botones disponibles

  • Insertar: Registra un nuevo servicio con los datos introducidos.
  • Modificar: Permite modificar la hora de fin de un servicio existente.
    Para modificar un servicio, realiza doble clic sobre él en la tabla.
  • Eliminar: Elimina el servicio seleccionado previa confirmación.
  • Cancelar: Limpia el formulario y restablece el estado inicial.

Tabla de Servicios

En la parte inferior se muestra una tabla con todos los servicios registrados por el trabajador actual.

  • Obra
  • Fecha del servicio
  • Hora de inicio
  • Hora de fin

Para editar o eliminar un servicio, haz doble clic sobre la fila correspondiente.

Validaciones y restricciones

  • La fecha debe estar completa y respetar el formato indicado.
  • La hora de fin debe ser posterior a la hora de inicio.
  • No se permite registrar servicios que se solapen en horario con otros servicios del mismo trabajador en la misma fecha.
  • Al modificar un servicio, no es posible cambiar la obra, la fecha ni la hora de inicio.
